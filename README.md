## Description
LSL1 or Linux Subsystem For Linux v1 is a tool to run other linux distros through docker containers



## Installation
Clone the repository:
```bash
git clone https://github.com/MOHAPY24/LSL1.git
cd LSL1
```
then run the wrapper script as sudo or root
```bash
sudo chmod +x lsl1
sudo ./lsl1 help
```
